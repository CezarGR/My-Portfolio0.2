Botões desativados usando o elemento se comportam um pouco diferente: O elemento não suport o atributo disabled, então, você deve usar a classe .disabled para fazê-lo parecer desativado, visualmente; Alguns estilos (que ainda não são bem suportados) são usados para desativarem todos os pointer-events, em botões âncora; Em browsers que suportam essa propriedade, você não verá o cursor desativado. Botões desativados devem ter o atributo aria-disabled="true" para indicar o estado do elemento para tecnologias assistivas.
